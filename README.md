# Projet-ANDO-rapport---Optimisation-de-l-quipe-de-France-de-cyclisme
Rapport du projet Ando sur l’analyse de données et l’optimisation appliquées au cyclisme, incluant la modélisation multicritère et l’aide à la décision pour la sélection de l’équipe de France pour les championnats du monde 2025 à Kigali.
